# Design summary

- Design concept: Local hiring map
- Layout personality: split-screen local guide
- Typography: Bree Serif headings with Lato body text
- Color palette: primary #14532d, secondary #86efac, accent #f97316, background #f7fee7, text #1c1917, muted #64748b
- Shape language: Pill-shaped elements
- Visual motif: map pins and neighborhood chips
- Unique layout choices: first sections use Local coverage map, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
